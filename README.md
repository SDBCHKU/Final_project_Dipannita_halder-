# Final_project_Dipannita_halder
Student Cyber Security Management System

Objective

The objective of this project is to build a menu-driven application that manages student records along with basic cyber security assessment.
This project combines all the core concepts learned during the internship such as file handling, functions, loops, and problem-solving.


Features

* Add Student (Name, ID, Branch, Email)
* View All Students
* Search Student (by Name or ID)
* Delete Student Record
* Security Assessment System
* Generate Report (Average Score, Poor Security)
* File Storage using students.txt
* Password Strength Checker
* Username Generator
* Login Validation System


Technologies Used

* Python
* File Handling (Text File)
* Basic Cyber Security Concepts


Program Flow

1. Display Main Menu
2. User selects an option
3. Perform operation (Add, View, Search, Delete)
4. Save or retrieve data from file
5. Perform security assessment
6. Generate report
7. Repeat until Exit
